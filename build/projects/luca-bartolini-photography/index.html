Project: Luca Bartolini Photography

A friend of mine, who is a very talented amateur photographer, asked me to help him publishing a website about his pictures. He gave me a visual idea on how he wanted the website to look like, and I turned it into an elegant responsive website.

I've coded this website using pure HTML5, CSS and a bit of JavaScript. It's based on Middleman, a very powerful static site generator.
